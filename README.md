## Como começar?

```bash
# Instala as dependências
$ yarn install

# Servidor com start em localhost:3000
$ yarn dev

# buildar para produção e lançar o app
$ yarn build
$ yarn start

# Gera arquivos estáticos
$ yarn generate
```
